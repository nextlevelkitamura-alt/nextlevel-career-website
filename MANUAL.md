# Next Level Career - Project Manual

- **Local Development**

## 📂 Project Structure Note

- `app/`: Next.js App Router source code
- `components/`: React components
- `utils/supabase/`: Supabase client configuration
- `palns/`: Project planning and task tracking documents

## 🛠 Command Reference

- **Start Development Server**
  ```bash
  npm run dev
  ```

- **Deploy to Production**
  ```bash
  npx vercel deploy --prod
  ```
