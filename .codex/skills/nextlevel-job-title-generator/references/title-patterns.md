# Title Patterns

## Pattern Families

候補を作るときは、以下から最低6種類以上を混ぜる。同じ求人で同じ型を連続使用しない。

| 型 | 構造 | 例 |
|---|---|---|
| 高時給先頭型 | 時給/年収 + 職種 + エリア/条件 | 時給1600円！データ入力スタッフ／田町エリア・土日祝休み |
| 職種先頭型 | 職種 + 対象業務 + 条件 | 問い合わせ受付スタッフ／補助金申請サポート・平日17時まで |
| エリア先頭型 | 地名/駅 + 職種 + 条件 | 渋谷駅徒歩5分のカスタマーサポート／服装・髪型自由 |
| 条件先頭型 | 土日祝休み/週3可/在宅予定 + 職種 | 土日祝休みのスマホ検品スタッフ／大井競馬場前エリア |
| 業務具体型 | 具体業務 + 職種 + 給与 | Web申込データの確認・登録事務／時給1500円・秋葉原 |
| 安心材料型 | 研修/OJT/マニュアル + 職種 | OJTありの端末チェック作業／芝浦ふ頭・週3日から相談可 |
| 働き方型 | 勤務時間/休日 + 業務 | 9時-17時勤務の補助金問い合わせ受付／品川駅近く |
| 経験者訴求型 | 経験/スキル + 高条件 | 法人営業経験を活かす通信サービス支援／時給2100円 |
| 未経験寄せ型 | 未経験に言える根拠 + 職種 | マニュアル完備のヘアケア商品窓口／渋谷・交通費全額 |
| 黙々作業型 | 集中作業 + 対象物 + 条件 | 黙々進めるスマホ検品・梱包作業／土日祝休み |
| 公共系安心型 | 公共/自治体/補助金 + 受付/事務 | 公共系の補助金申請受付スタッフ／平日のみ・品川駅近く |
| 事務スキル型 | Excel/入力/メール + 業務 | Excelを使うバックオフィス事務／電気・ガス契約サポート |
| 夜勤帯明示型 | 遅番/夜帯 + 業務 + 注意 | 15時開始の問合せサポート／モバイルバッテリーサービス |
| 短期明示型 | 期間 + 職種 + 条件 | 7月末までの社内電話取次ぎ／新宿駅近く・時給1600円以上 |
| 週日数訴求型 | 週3/週5 + 職種 + 給与 | 週3日から相談できる携帯端末検品／時給1450円 |
| 交通費型 | 交通費全額 + 職種 + エリア | 交通費全額支給の店舗確認コール／渋谷エリア・平日勤務 |

## Attractive Fact Bank

求人票に記載がある場合だけ使う。

- 給与: `時給XXXX円`, `時給XXXX円以上`, `交通費全額支給`, `高時給`
- 場所: `駅徒歩X分`, `駅近く`, `都心エリア`, `渋谷`, `新宿`, `品川`, `田町`, `虎ノ門`
- 働き方: `土日祝休み`, `平日のみ`, `週3日から相談可`, `週5日`, `17時まで`, `7.5時間勤務`
- 環境: `服装自由`, `髪型自由`, `ネイル自由`, `オフィスカジュアル`
- 安心材料: `研修あり`, `OJTあり`, `マニュアルあり`, `受電のみ`, `事務メイン`
- 仕事内容: `データ入力`, `問い合わせ受付`, `カスタマーサポート`, `営業サポート`, `端末検品`, `キッティング`, `書類処理`, `架電`, `受発注`

## Avoid

- 「誰でも簡単」「絶対残業なし」「完全在宅」「正社員になれる」など、確約できない表現。
- 派遣先企業名の無断掲載。
- 求人票の開始日が過去なのに「即日スタート」と断定すること。
- 成人向け商材など、応募前告知が必要な要素を隠しすぎること。
- 同じ一覧内で `時給XXXX円！〇〇スタッフ／〇〇エリア・交通費全額` を繰り返すこと。

## Batch Diversity Rules

複数求人を一括で扱う場合は、以下を守る。

- 1求人目を給与先頭にしたら、2求人目は職種先頭、3求人目はエリア先頭などにずらす。
- 同じエリアが続く場合は、片方を仕事内容先頭、片方を条件先頭にする。
- 同じ職種カテゴリが続く場合は、業務対象を具体化して差を出す。
- 推奨Top3は、強い順ではなく「一覧で並べたときに被らない順」も考慮する。
