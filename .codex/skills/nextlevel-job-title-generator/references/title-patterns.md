# Title Patterns

## NextLevel Dispatch Tone

既存の派遣求人は、長いコピーよりも「駅名・エリア」「業務名」「時給・勤務条件」を読みやすく並べる掲載型が強い。派遣タイトルは、まずこの温度感に寄せる。

参考にする既存型:

- `【溜池山王】スマホ初期設定のWeb接客サポート｜時給1,500円・週3日OK・交通費全額支給`
- `【池袋】携帯契約・料金の問い合わせ対応｜時給1,460円・週3日OK・交通費全額支給`
- `【御茶ノ水】書籍通販カスタマーサポート｜時給1,500円・土日祝休み・交通費全額支給`
- `週4日OKのアパレル・雑貨ECのカスタマーサポート｜渋谷区勤務・時給1,460円・交通費全額支給`
- `土日祝休みの受発注・問い合わせ対応事務｜品川区勤務・時給1,460円・交通費全額支給`
- `法人向け携帯サービスの問い合わせ対応募集｜新宿区エリア・時給1,600円・土日祝休み`

この型に合わせるときは、先頭を煽りすぎない。`高時給！` や `急募！` を連発せず、業務内容が一瞬で分かる名詞句を作る。

## Pattern Families

候補を作るときは、以下から最低6種類以上を混ぜる。同じ求人で同じ型を連続使用しない。

| 型 | 構造 | 例 |
|---|---|---|
| 既存掲載寄せ型 | `【駅名】業務名｜時給・条件・交通費` | 【秋葉原】Web申込データの確認・登録事務｜時給1,500円・交通費全額支給 |
| エリア勤務型 | `業務名｜エリア勤務・時給・条件` | 補助金申請の問い合わせ受付｜品川駅近く・時給1,460円・土日祝休み |
| 条件自然文型 | `条件の業務名｜エリア・給与` | 土日祝休みのスマートフォン検品・キッティング｜大井競馬場前・時給1,500円 |
| 業務具体型 | `対象物 + 作業/対応 + 職種｜エリア` | 電気・ガス契約のチェック事務｜秋葉原エリア・交通費全額支給 |
| 職種明快型 | `職種/業務名募集｜エリア・条件` | 法人向け通信サービスの営業サポート募集｜虎ノ門エリア・時給2,100円 |
| 週日数型 | `週3日OK/週4日OK + 業務名｜エリア` | 週3日から相談できる携帯端末検品｜芝浦ふ頭・交通費全額支給 |
| 公共系型 | `公共/補助金/自治体 + 受付/事務｜条件` | 補助金申請の受付・事務スタッフ｜品川駅近く・平日17時まで |
| カスタマー型 | `商材/サービス + カスタマー窓口｜条件` | ヘアケア商品のカスタマー窓口｜渋谷駅徒歩5分・服装自由 |
| コール方向明示型 | `問い合わせ対応/店舗確認コール/受電のみ｜条件` | モバイルバッテリー設置店舗への確認コール｜渋谷勤務・服装自由 |
| 軽作業型 | `検品/キッティング/回送 + スタッフ｜条件` | 携帯端末の検品・データ入力スタッフ｜芝浦ふ頭・週3日から相談可 |
| 短期型 | `短期/期間 + 業務名｜エリア・時給` | 7月末までの社内電話取次ぎ｜新宿駅近く・時給1,600円以上 |
| 経験活用型 | `経験を活かせる + 業務名｜給与` | 法人営業経験を活かせる通信サービス支援｜虎ノ門・時給2,100円 |

## Output Mix

単発案件では以下の比率を目安にする。

- 既存掲載寄せ: 4案
- 業務具体・職種明快: 4案
- 条件自然文・週日数・公共系など: 3-5案
- 少し訴求強め: 1-3案

一括案件では、各求人のTop1だけでも先頭語が散るようにする。

## Attractive Fact Bank

求人票に記載がある場合だけ使う。

- 給与: `時給1,500円`, `時給1,600円以上`, `交通費全額支給`, `高時給`
- 場所: `駅徒歩5分`, `駅近く`, `渋谷`, `新宿`, `品川`, `秋葉原`, `田町`, `虎ノ門`
- 働き方: `土日祝休み`, `平日のみ`, `週3日から相談可`, `週4日OK`, `週5日`, `17時まで`
- 環境: `服装自由`, `髪型自由`, `ネイル自由`, `オフィスカジュアル`
- 安心材料: `研修あり`, `OJTあり`, `マニュアルあり`, `受電のみ`, `事務メイン`
- 仕事内容: `データ入力`, `問い合わせ受付`, `カスタマーサポート`, `営業サポート`, `端末検品`, `キッティング`, `書類処理`, `架電`, `受発注`, `車両回送`

## Avoid

- 勤務時間を先頭に置くこと。`9時-17時勤務の...` ではなく `補助金申請の受付・事務｜品川駅近く・平日17時まで` のように後ろへ置く。
- `時給XXXX円！〇〇スタッフ／〇〇エリア・交通費全額` の同一テンプレを続けること。
- 「誰でも簡単」「絶対残業なし」「完全在宅」「正社員になれる」など、確約できない表現。
- 派遣先企業名の無断掲載。
- 求人票の開始日が過去なのに「即日スタート」と断定すること。
- 成人向け商材など、応募前告知が必要な要素を隠しすぎること。

## Batch Diversity Rules

複数求人を一括で扱う場合は、以下を守る。

- 1求人目を `【駅名】` にしたら、2求人目は業務名先頭、3求人目は条件先頭などにずらす。
- 同じエリアが続く場合は、片方を仕事内容先頭、片方を条件先頭にする。
- 同じ職種カテゴリが続く場合は、業務対象を具体化して差を出す。
- 推奨Top3は、強い順だけではなく「一覧で並べたときに被らない順」も考慮する。
