## 2. サイトの更新方法
今後、色を変えたり文字を直したりした場合の更新作業はとても簡単です。

1.  **コードを修正する** (VS Codeで保存)
2.  **GitHubへ保存する** (以下のコマンドを実行)
    git commit -m "修正内容のメモ"
    git push
    ```
3.  **自動更新**
    *   GitHubへの保存(`git push`)が完了すると、Vercelがそれを検知して**自動的に新しいバージョンをデプロイ**し始めます。
    *   数分待てば、同じURLのまま中身だけが最新版に切り替わります。

これだけです！面倒なサーバー操作は必要ありません。
