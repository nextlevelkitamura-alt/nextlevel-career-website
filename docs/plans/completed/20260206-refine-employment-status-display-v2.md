# 雇用形態表示の改善計画 V2

## 概要
ユーザーフィードバックに基づき、雇用形態表示をさらに改善します。

## 変更点

### 1. 「雇用形態」という文字（ラベル）の削除
- 現在: `雇用形態 : 正社員`
- 変更後: `正社員`
- 対象: 求人カード、求人詳細ページ

### 2. 「正社員」の水色表示の修正
- 前回の変更で `sky-600` を指定しましたが、反映されていない（黒文字になっている）ようです。
- 原因として、Tailwindの設定で `sky` カラーが有効でないか、文字列判定の問題が考えられます。
- **対策**:
  - カラーパレットを標準的な `blue` (または `cyan`) に変更し、確実に色がつくようにします。 (`text-blue-500` 等)
  - 文字列判定を部分一致 (`includes`) に変更し、表記ゆれ（空白など）にも強くします。

## 実装手順
1. `lib/utils.ts` の `getEmploymentTypeStyle` 関数を修正。
2. `components/JobCard.tsx` の表示テキストを修正。
3. `app/jobs/[id]/page.tsx` の表示テキストを修正。
4. デプロイして確認。
