# 派遣カラーの明確化計画

## 概要
「派遣」の雇用形態およびタグの色を、明確に**ピンク色**として定義します。

## 変更内容
- `lib/utils.ts` のロジックにおいて、「派遣」が含まれる場合を明示的にピンク色にする処理を確認・追加します。
  - 現状もデフォルトでピンクですが、明示的に記述することで意図しない変更を防ぎます。
  - 色コードは既存のタグと同じピンク (`pink-600`) を使用します。

## 実装イメージ
```typescript
if (type.includes("正社員")) {
    return Blue; // 水色
}
if (type.includes("派遣")) {
    return Pink; // ピンク（この部分を保証）
}
return Pink; // その他デフォルト
```
