# 雇用形態表示の改善計画

## 概要
求人カードおよび求人詳細ページにおける「雇用形態」の表示を改善し、視認性を向上させます。現状のグレーのテキスト表示から、ユーザー提供の参考画像に近い「ピンクの枠線・ラベル付き」のデザインに変更します。

## 変更内容

### 1. 求人カード (JobCard.tsx)
- 現在の `job.type` の単純なテキスト表示を廃止します。
- 新しいデザインコンポーネントに置き換えます。
  - スタイル: ピンク色の枠線 (`border-pink-200`)、白背景、ピンク色の文字
  - 表示フォーマット: `雇用形態 : {job.type}`
  - 配置: カテゴリの下、タイトルの上（現状の位置を維持しつつ強調）

### 2. 求人詳細ページ (JobDetailPage)
- ヘッダー部分（タイトル上部）のタグ表示をカードのデザインと統一感のあるものに変更します。
- `雇用形態` というラベルを明示するか、色味を合わせて強調します。

## 実装ファイルの場所
- `components/JobCard.tsx`
- `app/jobs/[id]/page.tsx`

## 検証手順
1. トップページまたは求人一覧ページを開き、求人カードの「雇用形態」がピンク色の枠で囲まれ、目立つようになっているか確認します。
2. 求人詳細ページに遷移し、ヘッダー部分の雇用形態表示が意図通りに変更されているか確認します。
