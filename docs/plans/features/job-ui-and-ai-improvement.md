# アイデア: 求人UIとAI登録の改善

## 概要

既存機能のユーザー体験向上を目的とした改善

### 求人の見え方改善
- **カードUIのデザイン改善**: アイコン、色、レイアウトの最適化
- **詳細ページの情報構成**: 情報の順序や見せ方の改善
- **おすすめ表示の精度向上**: パーソナライズやアルゴリズムの改善

### AI登録の改善
- **抽出精度の向上**: 求人原稿からの情報抽出の正確性向上
- **チャット修正の改善**: 自然言語での修正指示の処理改善
- **新規入力の支援**: ゼロから求人を作る場合のAIアシスト機能
- **バッチ処理の改善**: 一括登録時の処理や確認フローの改善

## 必要なサービス
- Google Gemini 2.0 Flash（既存）

## 必要な環境変数
- 既に設定済み（GOOGLE_API_KEY）

## 想定される実装内容

### 求人の見え方
1. カードUIのリデザイン
   - アイコンの追加・統一
   - カラースキームの最適化
   - 情報の階層化
2. 詳細ページの情報構成見直し
   - 重要情報の優先表示
   - スクロール体験の改善
3. おすすめアルゴリズムの改善
   - ユーザー属性に基づくパーソナライズ
   - 閲覧履歴の活用

### AI登録の改善
1. 抽出精度の向上
   - プロンプトエンジニアリングの改善
   - フォーマットバリデーションの強化
2. チャット修正の改善
   - コンテキスト理解の向上
   - 複数指示の同時処理
3. 新規入力の支援
   - テンプレート提案機能
   - 項目自動補完
4. バッチ処理の改善
   - 進捗表示の改善
   - エラーハンドリングの強化

## 優先度
中（既存機能の改善・拡張）
