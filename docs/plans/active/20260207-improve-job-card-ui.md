# 求人カードUI改善プラン

## 目的
求人カードと詳細ページでの雇用形態表示とタグの色統一感を改善する

## 現状
- 正社員: 青色（`text-blue-600`, `border-blue-200`）
- 派遣: ピンク色（`text-pink-600`, `border-pink-200`）※コード上は実装済みだが表示に問題あり
- タグ: 雇用形態に応じた色付け

## 問題点
1. 派遣の文字に色がついていない（ユーザー報告）
2. タグの色の統一感が不足している可能性

## 改善内容

### 1. 雇用形態バッジの色統一
- 正社員: 青系統（`bg-blue-50`, `text-blue-600`, `border-blue-200`）
- 派遣: ピンク系統（`bg-pink-50`, `text-pink-600`, `border-pink-200`）

### 2. タグの色統一
- 正社員の求人: 青系統のタグ
- 派遣の求人: ピンク系統のタグ

### 3. 対象ファイル
- `components/JobCard.tsx` - 求人一覧カード
- `app/jobs/[id]/page.tsx` - 求人詳細ページ
- `lib/utils.ts` - スタイルユーティリティ関数

## 実装方針
- `/gemini` でUI実装をGemini 3.0 Proに引き継ぐ
- 既存の `getEmploymentTypeStyle` と `getJobTagStyle` 関数を確認・修正

## 受入テスト
1. 正社員の求人カードで青色が表示される
2. 派遣の求人カードでピンク色が表示される
3. タグの色が雇用形態に合わせて統一されている
