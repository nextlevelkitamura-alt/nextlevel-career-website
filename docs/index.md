## 企画ライフサイクル

1. **アイデア** → `企画/` に作成（状態: アイデア）
2. **提案** → 仕様・工数を追記（状態: 提案）
3. **採用** → `企画/採用/` に移動 → ROADMAP.md に週割当
4. **計画化** → `計画/未着手/` または `計画/進行中/` に移動
5. **実装** → `計画/進行中/` で作業（状態: 進行中）
6. **完了** → `計画/完了/` に移動（状態: 完了）
