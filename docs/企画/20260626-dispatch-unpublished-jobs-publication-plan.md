# 派遣求人未掲載候補の掲載立案

## 背景・課題

デスクトップの `未掲載候補_派遣求人票_20260626` にある派遣求人票PDF 18件を、NextLevel Career Site に掲載する前提で整理する。

求人票には、開始日が過去のもの、終了予定が近いもの、求人名や就業場所が空欄のもの、公開表現に注意が必要な商材を扱うものが混在している。掲載作業に入る前に、初回掲載向きの案件と確認待ち案件を分ける。

## 提案内容

### 掲載方針

- 初回は「関東圏」「長期または随時開始」「給与・勤務条件が明確」「求職者に説明しやすい」案件を優先する。
- 求人票内の担当者名、個人連絡先、元帳票名、社内向け表現は公開求人に出さない。
- クライアント企業名は、公開許可が明確な場合だけ掲載する。迷う案件は「大手通信グループ関連」「公共系窓口」「エネルギー関連センター」などに匿名化する。
- `1900年1月0日` は長期扱いの入力値とみなし、公開文面では使わない。
- 2026年6月26日時点で開始日が過去の案件は、募集継続確認後に「即日」「随時」に更新する。
- 派遣求人として、雇用形態は原則 `派遣`、給与は `時給 + 交通費全額支給`、休日・勤務時間・服装条件を明確に出す。
