## 🎯 タイトルの打ち出し方

### 派遣社員のタイトル例

**基本パターン:**
```
【時給{金額}円】【{訴求タグ}】{職種}
```

**具体例:**
- `【時給1500円】【未経験OK】一般事務@六本木駅直結`
- `【時給1800円】【即日スタート】データ入力@新宿`
- `【時給1600円】【交通費全額】受付事務@大手メーカー`
- `【時給1700円】【服装自由】コールセンター@池袋`

**重視する要素（優先順）:**
1. **時給** — 最も目立つように最初に配置
2. **訴求タグ** — 未経験OK、即日スタート、交通費全額、残業なし、服装自由など
3. **職種** — シンプルに
4. **最寄駅** — アクセスの良さを訴求（オプション）

---

### 正社員のタイトル例

**基本パターン:**
```
【{訴求タグ}】{職種} | {企業の特徴}
```

**具体例:**
- `【年収400万円〜600万円】Webエンジニア | 成長中のSaaS企業`
- `【リモートワーク可】営業職 | 大手IT企業`
- `【未経験歓迎】ITサポート | 研修充実・年間休日125日`
- `【マネージャー候補】事業企画 | 東証プライム上場企業`

**重視する要素（優先順）:**
1. **年収レンジ** — 給与の透明性
2. **訴求タグ** — リモートワーク可、年間休日125日、未経験歓迎、残業月20時間以下など
3. **職種** — 具体的に
4. **企業の特徴** — 成長企業、大手、上場企業、業界No.1など

## 🤖 AI最適化の方針

### 派遣社員向けAIプロンプト

#### 抽出で重視すべき項目（優先順）:
1. **時給** — 数値を正確に抽出
2. **就業先企業名** — **⚠️ 基本的に匿名化（非公開）する**
   - AI抽出時に企業名が明記されていても、自動的に業種・規模で代替
   - 例: 「株式会社○○」→「大手メーカー」「IT企業」「外資系コンサル」など
   - `is_client_company_public` は **デフォルトで `false`**
3. **勤務時間・実働時間** — 「9:00〜18:00（実働8時間）」
4. **交通費** — 全額支給、上限あり、支給なし
5. **服装・髪型・ネイル規定** — オフィスカジュアル、私服OK、ネイルOKなど
6. **即日スタート** — 開始時期の柔軟性
7. **勤務期間** — 長期、短期、3ヶ月更新など
8. **週の勤務日数** — 週5日、週3日〜相談可など

#### 企業名の匿名化ルール:

**⚠️ 重要: 派遣求人は基本的に企業名を非公開**

AI抽出時に企業名が明記されている場合、以下のように自動匿名化：

| 元の企業名 | 業種・特徴から判断 | 匿名化後の表現 |
|-----------|------------------|---------------|
| トヨタ自動車株式会社 | 自動車メーカー | 大手自動車メーカー |
| 株式会社NTTデータ | IT・システム開発 | 大手IT企業 |
| ゴールドマン・サックス | 外資系金融 | 外資系金融機関 |
| 株式会社ファーストリテイリング | 小売・アパレル | 大手アパレル企業 |
| 不明・記載なし | 業種のみ記載 | {業種}企業（例: IT企業、メーカー） |

**匿名化の判定基準**:
- `is_client_company_public` が `false` の場合（デフォルト）
- 企業名が具体的に記載されている場合も、自動的に業種・規模に置き換える

#### タイトル生成ロジック:
```
【時給{hourly_wage}円】【{主要訴求タグ1〜2個}】{職種}@{最寄駅 or エリア}
```

**⚠️ 企業名はタイトルに含めない**（匿名化のため）

#### 訴求タグの自動生成ルール:
- `commute_allowance` に「全額」→ `交通費全額`
- `start_date` に「即日」→ `即日スタート`
- `overtime_hours` が「なし」→ `残業なし`
- `attire_type` が「自由」→ `服装自由`
- `requirements` に「未経験」→ `未経験OK`
- `nail_policy` に「自由」→ `ネイルOK`

---

### 正社員向けAIプロンプト

#### 抽出で重視すべき項目（優先順）:
1. **年収レンジ** — `annual_salary_min` 〜 `annual_salary_max`
2. **企業名・業界** — 企業のブランド力・業界
3. **企業概要・事業内容** — 企業の魅力
4. **仕事の醍醐味・やりがい** — モチベーション要素
5. **残業時間** — ワークライフバランス
6. **年間休日** — 125日以上は大きな訴求ポイント
7. **リモートワーク・フレックス** — 柔軟な働き方
8. **キャリアパス・昇進制度** — 成長機会

#### タイトル生成ロジック:
```
【{主要訴求タグ}】{職種} | {企業の特徴}
```

#### 訴求タグの自動生成ルール:
- `annual_salary_min` が設定されている → `年収{min}万円〜{max}万円`
- `annual_holidays` >= 120 → `年間休日{days}日`
- `overtime_hours` が「月20時間以下」→ `残業月20時間以下`
- `requirements` に「未経験」→ `未経験歓迎`
- `benefits` に「リモート」→ `リモートワーク可`
- `probation_period` が「なし」→ `試用期間なし`
